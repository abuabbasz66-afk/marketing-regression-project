Marketing Regression Project

Project Overview

This project analyzes a marketing dataset using Simple Linear Regression to determine which advertising channel has the strongest impact on sales.

Dataset Features

- TV Advertising
- Radio Advertising
- Social Media Advertising
- Sales

Methodology

1. Data loading and cleaning
2. Exploratory Data Analysis (EDA)
3. Correlation analysis
4. Simple Linear Regression using Statsmodels OLS
5. Model evaluation and diagnostics

Key Findings

- TV advertising showed the strongest correlation with Sales.
- Correlation between TV and Sales was approximately 0.999.
- The regression model achieved an R-squared value of 0.999.
- TV advertising was statistically significant with a p-value less than 0.05.

Business Recommendation

Based on the analysis, TV advertising has the strongest impact on sales performance. Organizations should prioritize TV advertising investments when allocating marketing budgets to maximize return on investment (ROI).
